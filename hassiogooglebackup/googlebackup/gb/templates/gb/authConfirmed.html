Backup Instructions

The backup of your hass.io snapshots is performed by executing a REST service hosted by this add-on at the host port configured in the details for this add-on. There are no arguments needed to passed to the service request. Everything needed is configured in the add-on configs. The address for this service is in the form: https://hassio.local:[HOST_PORT]/gb/doBackup where [HOST_PORT] is the host port number you configured for this add-on (the same port number this page is served from).

Example: https://hassio.local:8055/gb/doBackup
